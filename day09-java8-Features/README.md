# 🚀 Day 09 – Java 8 Features

## 🔹 Introduction

Java 8 introduced powerful features that make code more concise, readable, and functional.
It is widely used in modern Java development and frameworks like Spring Boot.

---

## 🔹 Topics Covered

* Lambda Expressions
* Functional Interfaces
* Stream API
* Method References
* Optional Class
* Default & Static Methods
* Date & Time API

---

## 🔹 Lambda Expressions

Used to write shorter and cleaner code by replacing anonymous classes.

---

## 🔹 Functional Interface

An interface with only one abstract method.
Example: Runnable, Comparator

---

## 🔹 Stream API

Used to process collections of data in a functional style.

### Common Operations

* filter()
* map()
* forEach()
* collect()

---

## 🔹 Method Reference

Used to refer to methods directly using `::` operator.
Example: `System.out::println`

---

## 🔹 Optional Class

Used to avoid NullPointerException.
Provides methods like `orElse()`, `isPresent()`.

---

## 🔹 Default & Static Methods

* Default methods → allow method implementation inside interface
* Static methods → can be called using interface name

---

## 🔹 Date & Time API

* LocalDate
* LocalTime
* LocalDateTime

👉 Improved and thread-safe date handling.

---

## 🔹 Key Benefits

* Less code (concise syntax)
* Improved readability
* Better performance with streams
* Functional programming support

---

## 🔹 Practice Programs

* Filter even numbers using stream
* Find max/min using stream
* Sort list using lambda
* Convert list to uppercase
* Use Optional to handle null

---

## 🔹 Conclusion

Java 8 features simplify coding and improve performance. They are essential for modern Java development and frequently asked in interviews.
