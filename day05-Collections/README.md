# 📚 Day 07 – Java Collections Framework

## 🔹 Introduction

Java Collections Framework (JCF) is used to store and manage groups of objects dynamically.
It is part of the `java.util` package.

---

## 🔹 Topics Covered

* List Interface
* Set Interface
* Queue Interface
* Map Interface
* Iteration Techniques
* Differences & Use Cases

---

## 🔹 Collection Types

### 📌 List (Ordered, allows duplicates)

* ArrayList
* LinkedList
* Vector

### 📌 Set (No duplicates)

* HashSet
* LinkedHashSet
* TreeSet

### 📌 Queue (FIFO)

* PriorityQueue

### 📌 Map (Key-Value pairs)

* HashMap
* LinkedHashMap
* TreeMap
* Hashtable

---

## 🔹 Important Methods

### List

* add()
* get()
* set()
* remove()
* size()

### Set

* add()
* contains()
* remove()

### Map

* put()
* get()
* remove()
* containsKey()
* keySet()

---

## 🔹 Iteration Methods

* For loop
* Enhanced for loop
* Iterator
* forEach (Java 8)

---

## 🔹 Key Differences

* List → allows duplicates
* Set → no duplicates
* Map → key-value pairs
* HashMap → fast, no order
* TreeMap → sorted data

---

## 🔹 Use Cases

* ArrayList → fast data access
* LinkedList → frequent insert/delete
* HashSet → unique elements
* HashMap → fast key-value storage

---

## 🔹 Practice Programs

* Remove duplicates from list
* Sort elements
* Count frequency using HashMap
* Find second largest element

---

## 🔹 Conclusion

Java Collections are essential for handling data efficiently and are widely used in real-world applications and interviews.
